# docs/assets — изображения для документации

Хранилище картинок, на которые ссылается документация (README и docs/*).

## Что лежит

- `screenshot-main.png` — скриншот главного окна FFNodes (превью + холст + каталог + командная строка). Используется в корневом [README.md](../../README.md).

## Правила

- Имена файлов — латиницей, в kebab-case, с понятным префиксом (`screenshot-…`, `diagram-…`).
- Ссылки из README — относительными путями (`docs/assets/имя.png`).
- Добавил картинку — впиши строку в этот список.
