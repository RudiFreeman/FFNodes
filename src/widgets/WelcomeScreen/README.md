# WelcomeScreen — приветственный экран

Первая точка входа в FFNodes при холодном старте (Спринт 5). Вместо сразу пустого холста
показывает экран с выбором: **Новый проект** / **Открыть проект** + список **последних
проектов**. После любого действия App переключается на холст (до конца сессии назад не
возвращаемся — решение по UX-минимализму).

## Состав

- `WelcomeScreen.tsx` — чистый презентационный компонент. Состояния не держит, всю механику
  получает колбэками из `App`:
  - `onNew` — создать новый проект (чистый холст input→output);
  - `onOpen` — открыть проект через системный диалог;
  - `onOpenRecent(path)` — открыть недавний по пути;
  - `recent` — список последних проектов (тип `RecentProject` из `../TopBar/RecentMenu`).

## Переиспользование

Ничего не дублирует: список недавних — тот же `useRecentProjects`, что и в выпадашке
«Недавние» (TopBar); открытие/создание — те же `useProject` / `useGraph.loadGraph`. Гейт
показа («приветствие или холст») живёт в `src/App.tsx` через флаг `started`.

## Тесты

Автотеста компонента нет — в проекте отсутствует Testing Library/jsdom (все фронт-тесты
покрывают чистые функции). Компонент презентационный, чистой логики не вносит. Пробел
зафиксирован в `tests/README.md` (раздел «🔴 Нужно покрыть»). Проверяется вживую.
